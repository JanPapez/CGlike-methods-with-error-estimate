# CGlike-methods-with-error-estimate
CG and CG-like methods with adaptive error estimate based on paper [Papez, Tichy: Estimating the error in CG-like algorithms for least-squares and least-norm problems, 2024], DOI: https://doi.org/10.1007/s11075-023-01691-x

An example how to run the methods is given in "run_example.m"
